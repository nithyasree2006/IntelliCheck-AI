IntelliCheck AI

An AI-powered plagiarism detection system that compares text using semantic similarity and detects AI-generated or paraphrased content.

#Features

- User Signup & Login
- AI-powered Plagiarism Detection
- Semantic Similarity Analysis
- AI Content Detection
- Paraphrase Detection
- File Upload (TXT, PDF, DOCX)
- Downloadable PDF Reports
- Report History
- H2 Database Integration

#Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Hugging Face API
- HTML
- CSS
- JavaScript
- Maven

#How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/nithyasree2006/IntelliCheck-AI.git
   ```

2. Open the project in IntelliJ IDEA.

3. Add your Hugging Face API token in:

   ```
   src/main/resources/application.properties
   ```

4. Run:

   ```
   PlagiarismcheckerApplication.java
   ```

5. Open:

   ```
   http://localhost:8080
   ```
